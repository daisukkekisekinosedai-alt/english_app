- 総回答数

- 苦手カテゴリ

## Render反映

```powershell
git add .
git commit -m "rename app and add nyan coach"
git push
```

Renderの自動デプロイがONなら自動反映されます。
手動なら Render → Manual Deploy → Deploy latest commit を押してください。
