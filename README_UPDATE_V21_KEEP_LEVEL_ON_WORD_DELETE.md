# にゃんにゃんイングリッシュ v21 Keep Level on Word Delete

## 修正内容

単語を削除するとレベル・EXPが下がる問題を修正しました。

## 原因

これまで、単語削除時に以下も一緒に削除していました。

```sql
DELETE FROM study_logs WHERE word_id IN (...)
```

レベル計算は `study_logs` を見ているため、
CSVバッチ削除や単語削除を行うと、過去の回答履歴が消え、
EXPやレベルが下がる状態でした。

## 修正後

単語を削除しても `study_logs` は削除しません。

保持されるもの:

- レベル
- EXP
- 連続学習
- 回答数
- 正解数
- リスニング回答数
- バッジ判定用の学習実績
- テストセッション履歴
- ランキング用のテスト履歴

削除されるもの:

- words の単語データ
- user_word_flags のお気に入り紐づけ

## 表示への影響

削除した単語は `words` から消えるため、単語一覧・ミスノート・復習対象には表示されません。

ただし、過去にその単語を解いた事実は `study_logs` に残るため、
レベルやEXPは下がりません。

## 注意

すでに過去版で削除してしまった `study_logs` は、DBバックアップがない限り復元できません。
この修正は、今後の削除でレベルが下がらないようにするものです。
