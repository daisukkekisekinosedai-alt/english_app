# にゃんにゃんイングリッシュ v16 Smart Study Engine

## 大型アプデ概要

見た目機能ではなく、学習効果を上げるための機能的アップデートです。

## 追加機能

### 1. スマート学習ページ

新規ページ `/smart` を追加しました。

- 今日やるべき単語を自動表示
- 復習優先 / 弱点候補 / 新規単語 / 忘却防止 / 定着確認を分類
- 今日のミッションを自動生成
- 苦手カテゴリ・苦手品詞を分析

### 2. おすすめ10問

`scope=smart` を追加しました。

```text
/session/start?mode=choice&scope=smart&count=10
```

通常のランダムではなく、以下の優先度で出題します。

- 復習タイミングの単語
- 何度も間違えている単語
- 正答率が低い単語
- 7日以上触れていない単語
- 新規単語
- 定着確認用単語

### 3. 出題順の最適化

スマート学習ではランダムシャッフルせず、
優先度スコアが高い順に出題します。

### 4. スコアリング

各単語にスコアを付けます。

- 未回答
- ミス回数
- 正答率
- 最後に解いた日
- 最後に間違えた日
- 復習間隔

### 5. にゃんコーチとの連携

10問結果画面で、スマート学習専用のコメントを出します。
