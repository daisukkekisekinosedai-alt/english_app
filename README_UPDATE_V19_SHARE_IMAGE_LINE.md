# にゃんにゃんイングリッシュ v19 Share Image + LINE

## 追加したこと

結果共有を、**スクショ前提** から **画像自動生成** へアップデートしました。

## できるようになったこと

### 1. テスト結果を画像化して共有
- 画面: `/share/session/<id>`
- 共有画像プレビューを表示
- 「画像を共有（LINEなど）」で共有シートを開く
- スマホなら LINE を選んで送れる想定

### 2. 今日の結果を画像化して共有
- 画面: `/share/today`
- その日の合計結果を画像にして共有できる

### 3. 保存ボタン
- 共有に未対応のブラウザでも画像保存できる
- 保存した画像を LINE で手動送信可能

## 技術的な仕様

- サーバー側で SVG を生成
- ブラウザ側で PNG に変換
- Web Share API 対応ブラウザではファイル共有
- 非対応ブラウザは画像保存へフォールバック

## 反映


iPhone / Android ではかなり相性が良いです。
