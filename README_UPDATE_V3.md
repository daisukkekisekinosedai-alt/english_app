# にゃんにゃんイングリッシュ Update v3

ユーザー要望対応版です。

## 追加内容

### 1. masterアカウント
- 最初に登録したユーザーを自動でmasterにします。
- 既存DBでmasterがいない場合、一番古いユーザーをmasterに昇格します。
- `MASTER_USERNAME=master` を環境変数で変えることもできます。

### 2. masterだけできること
- 単語追加
- 単語編集
- 単語削除
- CSVインポート
- サンプル単語追加
- 音声ファイル生成/管理
- ユーザー削除

通常ユーザーは学習・テスト・ランキング・共有・お気に入りのみ利用できます。

### 3. ユーザー一覧を非公開化
- 通常画面から「ユーザー一覧」を削除しました。
- `/users` にアクセスしてもランキングへ戻します。
- masterだけ `/admin/users` で管理画面を見られます。

### 4. 猫の種類
プロフィールから相棒ねこを選べます。

- ミルクねこ
- あそびねこ
- まったりねこ
- ねむねこ
- 勉強ねこ

### 5. 10問完走エフェクト
10問クイズ完了時、正答率80%以上なら紙吹雪・ハート・肉球エフェクトで祝います。
100%なら満点表示も出ます。

## Renderへの反映

GitHubリポジトリにこのzipの中身を上書きしてpushしてください。

```powershell
git add .
git commit -m "update master cat effects"
git push
```

Render側で自動デプロイされます。
手動なら Render の Manual Deploy → Deploy latest commit を押してください。

## 注意

このアップデートはDBに以下の列を自動追加します。

- users.role
- users.cat_type

既存DBはそのまま使えます。
