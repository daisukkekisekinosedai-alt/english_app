# にゃんにゃんイングリッシュ v13

## 方針変更

アクセサリー/着せ替え機能は廃止しました。

前回までのクローゼット・アクセサリー関連は見た目の品質が低かったため、
この版では **着せ替え機能を完全に入れていません**。

## ベース

安定していた以下の版をベースにしています。

- v11 にゃんにゃんイングリッシュ
- にゃんコーチ
- 最初の猫画像に差し戻し済み
- 猫の部屋あり

## 追加したこと

master アカウントの場合、猫の部屋の家具をすべて使用可能にしました。

### masterの挙動

- 家具がすべて解放済みになる
- 進捗がすべて 100% 表示になる
- 学習条件を満たしていなくても家具が部屋に表示される

### 一般ユーザーの挙動

- 今まで通り、学習実績で家具が解放される
- 条件未達の家具は未解放のまま

## DB変更

なし。

既存ユーザー、単語、学習履歴は消えません。
