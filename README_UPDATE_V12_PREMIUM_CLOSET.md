# にゃんにゃんイングリッシュ v12 Premium Closet 修正版

## 修正理由

前回v12の着せ替えは、絵文字を重ねただけで安っぽく見えました。
この版では、猫画像の上に重ねるアクセサリを **絵文字表示ではなくCSSで描画するアクセサリ** に変更しています。

## 改善内容

- 絵文字をそのまま重ねる方式を廃止
- リボン、王冠、メガネ、首輪、マフラー、羽、ケープ、メダルをCSSで描画
- クローゼット一覧にもアクセサリプレビューを表示
- 装備中/未解放/解放済みの見た目を改善
- 既存の猫画像は最初の猫のまま維持

## DB変更

前回v12と同じです。

追加テーブル:
- user_cat_inventory
- user_cat_equipment

既存ユーザー・単語・学習履歴は消えません。
