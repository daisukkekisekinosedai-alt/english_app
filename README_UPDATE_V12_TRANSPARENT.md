# にゃんにゃんイングリッシュ v12 透過調整版

## 変更内容

着せ替えアクセサリが強く見えすぎたため、
**猫本体になじむように全体をより透過**しました。

### 調整したこと

- 猫の上のアクセサリ全体を半透明化
- mix-blend-mode を使ってなじませる調整
- 影を弱めて「貼った感」を軽減
- メガネ/片眼鏡はさらに薄く調整
- 羽/ケープのように面積が大きいものはもっと薄く調整
- クローゼットのプレビューは見やすさを優先して少しだけ濃く維持
