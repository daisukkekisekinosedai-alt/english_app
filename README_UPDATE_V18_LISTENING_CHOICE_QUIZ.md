# にゃんにゃんイングリッシュ v18 Listening Choice Quiz

## 変更内容

リスニングテストを、入力式ではなく **リスニング4択問題** に変更しました。

## 変更前

```text
音声を聞く
↓
日本語訳を入力する
```

## 変更後

```text
音声を聞く
↓
日本語訳を4択から選ぶ
```

## 対応した画面

### 1問リスニング

URL:

```text
/listen
```

- 音声を聞く
- 英単語は最初は隠す
- 日本語訳を4択で選ぶ
- 「英単語を見る」はヒントとして残す

### 10問連続リスニング

URL例:

```text
/session/start?mode=listen&scope=all&count=10
```

- 10問連続でもリスニング4択になります
- 入力式ではなく4択ボタンで回答します

## 選択肢の作り方

通常の4択クイズと同じ `get_choices()` を使っています。

- 正解単語と同じ品詞の日本語訳を優先
- 足りない場合だけランダム補完
- 品詞バレしにくい選択肢になります

既存ユーザー、単語、学習履歴は消えません。
