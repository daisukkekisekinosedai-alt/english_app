# にゃんにゃんイングリッシュ Update v6

## 変更内容

### 猫リアクション強化

以下のタイミングで猫のリアクションと演出が変わります。

- 正解
- 不正解
- 3問連続正解
- 5問連続正解
- 10問連続正解
- 10問クイズ80%以上
- 10問クイズ満点
- ホーム画面の連続学習日数

## 具体例

```text
正解
→ 猫が喜ぶ / キラキラ

不正解
→ 猫がしょんぼり / 復習メッセージ

3問連続正解
→ 肉球拍手

5問連続正解
→ 紙吹雪

10問連続正解
→ 猫が祭り状態

10問満点
→ PERFECT表示 + お祭り演出
```

## ついでに修正

v5でサンプル単語追加時に品詞推定が未定義になる可能性があったため、そこも修正しています。
